# MEMBUAT MODEL NATURAL STYLE 

- Untuk code .ipynb ada di file tugas_per_2.ipynb
- Untuk image hasil dari generate menggunakan model Natural Style ada di folder image-hasil-generate/stylized_image_hq.jpg
- Untuk data image mentah/yang belum dilakukan prosess generate dengan model Natural Style ada di folder image-data-mentah/

## Cara menjalankan Code ipynb
- Download 2 image(gunung bromo.jpeg untuk gambar utama dan sun-flowers.jpeg untuk style), ada di dalam folder image-data-mentah/
- Buka file tugas_per_2.ipynb
- Klik Open in Colab
- Upload 2 image yang tadi sudah di download
- Copy path dari kedua image tersebut
- Lalu pastekan dibagian CONTOH PENGGUNAAN yang ada dibagian bawah dari file ipynb
- Sesuaikan path untuk foto utama dan foto style
- CONTENT_IMAGE_PATH = paste path dari image gunung bromo disini
- STYLE_IMAGE_PATH =  paste path dari image sun-flowers disini
- Run Program
